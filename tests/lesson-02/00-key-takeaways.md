## A. Viết file ở dạng markdown
### 1. Tiêu đề (Headers)
```
# Tiêu đề cấp 1
## Tiêu đề cấp 2
### Tiêu đề cấp 3
#### Tiêu đề cấp 4
##### Tiêu đề cấp 5
###### Tiêu đề cấp 6
```

### 2. Nhấn mạnh văn bản (Text Emphasis)
```
*Chữ nghiêng* hoặc _Chữ nghiêng_
**Chữ đậm** hoặc __Chữ đậm__
***Chữ đậm và nghiêng*** hoặc ___Chữ đậm và nghiêng___
~~Gạch ngang~~
```

### 3. Danh sách (Lists)
**Danh sách không thứ tự:**
```
- Mục 1
- Mục 2
  - Mục con 2.1
  - Mục con 2.2
- Mục 3
```
**Danh sách có thứ tự:**
```
1. Mục đầu tiên
2. Mục thứ hai
3. Mục thứ ba
   1. Mục con 3.1
   2. Mục con 3.2
```

### 4. Liên kết (Links)
```
[Văn bản hiển thị](https://example.com)
[Văn bản có tooltip](https://example.com "Đây là tooltip")
```

### 5. Hình ảnh (Images)
```
![Văn bản thay thế](đường-dẫn-hình-ảnh.jpg)
![Văn bản thay thế](đường-dẫn-hình-ảnh.jpg "Tiêu đề hình ảnh")
```

### 6. Trích dẫn (Blockquotes)
```
> Đây là một trích dẫn
> Có thể nhiều dòng
>
> > Trích dẫn lồng nhau
```

### 7. Code
**Code inline:**
```
Sử dụng `code` trong câu
```
**Code block:**
```
```javascript
function hello() {
console.log("Hello World!");
}```
```

### 8. Đường kẻ ngang (Horizontal Rules)
```
---
hoặc
***
hoặc
___
```

### 9. Bảng (Tables)
```
| Cột 1 | Cột 2 | Cột 3 |
|-------|-------|-------|
| Dữ liệu 1 | Dữ liệu 2 | Dữ liệu 3 |
| Dữ liệu 4 | Dữ liệu 5 | Dữ liệu 6 |

Căn chỉnh:
| Trái | Giữa | Phải |
|:-----|:----:|-----:|
| Text | Text | Text |
```

### 10. Checkbox (Task Lists)
```
- [x] Nhiệm vụ đã hoàn thành
- [ ] Nhiệm vụ chưa hoàn thành
- [ ] Nhiệm vụ khác
```
